HTML

  • HTML stands for Hypertext Markup Language.
  • Markup language used to create the structure of web pages. Uses tags to define elements like headings, paragraphs, links, and images.
  • HTML5 is the latest version of HTML.
  • Introduced new elements and attributes for modern web development. Supports multimedia, canvas, and offline capabilities.
  • Semantic HTML enhances accessibility and SEO.
  • Use semantic tags like header, nav, main, footer to improve document structure. Helps search engines understand content and improves screen reader navigation.

CSS

  • CSS stands for cascading style sheets.
  • A margin shows how much space we want around the outside of an element.
  • A padding idicates how much space we want around the content inside an element
  • Helps control the layout, colors, fonts, and other visual aspects of web pages
  • CSS rules consist of a selector and a declaration block. Selector targets HTML elements, and the declaration block contains property-value pairs enclosed in curly braces.

GIT

  • Git is a distributed version control system used to track changes in files and collaborate on software development projects.
  • Basic git commands are git add, git commit -m, git status, git pull, git push.
  • Git checkout switches to the specified branch and is used for Branching in Git
  • git pull and git push is used to fetch and commit changes in your local branch.

Javascript

  • javaScript is a high-level, interpreted programming language used for adding interactivity to web pages. It runs on the client-side (in the browser) and is widely used in web development.
  • avaScript code is enclosed within script tags in HTML or can be placed in a separate .js file and linked using script tags.
  • ariables store data. Declare them using let, const, or var. Data Types: string, number, boolean, null, undefined, object, array.
  • Arrays are used to store multiple values in a single variable.
  • Functions are blocks of reusable code that can be called with specific arguments.
  • for: Executes a block of code repeatedly for a specified number of times. while: Repeats a block of code while a given condition is true.
